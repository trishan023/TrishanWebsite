# trishantalukdar.com

Personal portfolio website for **Trishan Talukdar** — a data engineering and analytics professional based in Sydney, Australia with 7+ years of experience building scalable data platforms, modern data stacks, and AI-powered analytics solutions across Azure, GCP, and AWS.

Currently an Analytics Engineer at Pacific National, previously at Deloitte and Commonwealth Bank of Australia.

## Tech Stack

- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Hosting:** Cloudflare Workers
- **CI:** GitHub Actions
- **Domain:** trishantalukdar.com (Cloudflare)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Branching

- `dev` — development branch, all work happens here
- `main` — production, auto-deploys to trishantalukdar.com via Cloudflare
